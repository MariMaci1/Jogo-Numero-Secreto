<h1>🎮 Jogo do Número Secreto</h1>

<h2>Um jogo interativo e responsivo desenvolvido com o objetivo de praticar a lógica de programação em JavaScript e aplicando estilos modernos com CSS3.</h2>

<h3>🚀 Tecnologias Utilizadas</h3>
<p> 
* HTML5: Estruturação semântica da página.
* CSS3: Design moderno, responsivo e focado na experiência do usuário (UX).
* JavaScript : Lógica do jogo, geração de números pseudoaleatórios, controle de tentativas e manipulação dinâmica de eventos e textos na tela.
</p>

<h3>🧠 O que aprendi neste projeto</h3>
<p>
* Manipulação do DOM: Como conectar o JavaScript ao HTML para alterar textos, habilitar/desabilitar botões e reagir às ações do usuário em tempo real.
* Lógica e Condicionais: Implementação de estruturas de repetição e validações para garantir que o jogo funcione sem erros.
* Didática no Código: Organização e clareza para o código, estruturando funções com responsabilidade única e fáceis de ler, garantindo que a lógica seja compreensível para qualquer desenvolvedor que analise o repositório.
</p>
